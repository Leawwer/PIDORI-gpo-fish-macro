                     PIDORI GPO MACRO
///////////////////////////////////////////////////////////////
Привет! Это простая и понятная инструкция по установке, настройке и запуску макроса для рыбалки в Grand Piece Online (GPO). Макрос работает только на Windows, использует GUI на Tkinter, оверлей для детекции, Telegram-бот для уведомлений и команд. Если что-то пойдёт не так — опиши проблему мне, я помогу! Telegram — @saved_messenges.

Требования
ОС: Windows (из-за win32api и winsound).
Python: Версия 3.10 или выше (рекомендую 3.12).
Библиотеки: Установи из файла requirements.txt.

Полная инструкция
Шаг 1: Установка Python

Скачай Python с официального сайта: python.org/downloads.
Выбери версию 3.12+ для Windows.
Во время установки поставь галочку «Add Python to PATH» (чтобы команды работали в CMD).
Проверь: Открой командную строку (нажми Win + R, введи cmd, Enter) и набери python --version. Должно показать версию.

Шаг 2: Установка зависимостей

В CMD перейди в папку со скриптом: cd C:\путь\к\папке\со\скриптом.
Установи: pip install -r requirements.txt.
Если ошибка — попробуй python -m pip install -r requirements.txt.
Рекомендую запустить CMD от имени администратора.

Шаг 3: Запуск

В той же CMD от имени администратора с путём к вашей папке напишите команду python 1.py, что запустит сам макрос.

Шаг 4: Настройка Telegram-бота (опционально)
Если хочешь управлять макросом через Telegram (/check для статуса + скринов, /restart для перезапуска):

Создай бота в Telegram: Напиши @BotFather, следуй инструкциям — получи API-токен (типа 123456:ABC-DEF...).
Узнай свой Chat ID: Напиши @userinfobot — он покажет число (типа 123456789).
В программе: Запусти макрос, нажми «TgHook», введи токен и Chat ID, нажми «Save», открой «TgHook» снова и нажми на «Start».
Бот отправит уведомление при запуске: «Макрос запущен с ПК [имя_ПК]».

Команды:

/check — сделает скрин оверлея + весь экран, покажет, сколько запущен макрос, сколько он купил наживки и сколько сделал циклов заброса.
/restart — то же самое, что нажимать на F1 с компьютера, то есть через /restart ты можешь выключить макрос или включить его.

Кулдаун на /restart — 7 секунд.
Без Telegram макрос работает нормально! Это просто удобное дополнение для дистанционной работы с телефона.

Шаг 5: Настройка
Оверлей: Оверлей — это ключевой элемент макроса, который представляет собой полупрозрачный синий прямоугольник на экране. Он используется для захвата конкретной области экрана (индикатора рыбалки), где макрос анализирует цвета и движение бара. Это позволяет автоматически кликать и ловить рыбу на основе визуальной детекции.

Нажми F2 (по умолчанию) — покажет/скроет синий полупрозрачный прямоугольник. Перетаскивай/ресайзь для захвата зоны рыбалки в игре (ГУИ, когда ты вылавливаешь рыбу), чтобы не задевало ГУИ с наживками!
Для корректной работы надо правильно поставить камеру и оверлей — это очень важно. Оверлей должен быть на ГУИ с рыбой и !НАКАКИМ ОБРАЗОМ НЕ ЗАДЕВАТЬ ГУИ С НАЖИВКОЙ!. Надо подкорректировать, на это может уйти определённое время, чтобы понять, как правильно поставить камеру и оверлей.

Настройки: Изменяй Kp/Kd (механика, для стабильности рекомендую 2,2), тайминги.
Авто-покупка: Поставь английскую раскладку клавиатуры, укажи amount (количество наживки) и loops (циклы до покупки). Подойдите к наживке так, чтобы вы могли при нажатой E её купить (не удочку, а именно наживку). Лучше всего делать в трейдхабе.
Точки для авто-покупки:

Нажми E, стоя рядом с наживкой — откроется ГУИ в игре типа «хочешь ли ты купить?».
В макросе нажми:
«Set Pt 1», потом в игре нажми на кнопку Yes (в игре кнопка Yes нажаться не должна — это просто, чтобы поставить точки!).
«Set Pt 2» — ровно между Yes и No.
«Set Pt 3» — по кнопке No.
«Set Pt 4» — куда макрос будет забрасывать (просто в воду ближайшую).


Старт: Нажми F1. Кулдаун между нажатиями — 7 секунд (чтобы избежать случайных спамов).
Статистика: Purchases (покупки), Cycles (циклы), Uptime (время работы).
Сохранение: Нажми «Save cfg» — сохранит всё в cfg.json. При перезапуске само загрузит настройки, которые были сохранены, но бота надо будет запускать вручную, нажав на «Start».
Логи: Смотри в окне, очищай на «Clear», сохраняй на «Save Logs» (сохранять не обязательно — через сохранение можно понять, какие он ошибки делал, что он вообще делал всё это время).

Выход: F3 закроет все гуи идеально, без поломок и тд.

Возможные проблемы и решения:

Ошибки импорта: Проверь установку библиотек. Pywin32 — только для Windows.
Оверлей не работает: Проверь DPI-экрана (рекомендую 100% масштаб в настройках Windows).
Звуки не работают: Файлы .wav не найдены или не в формате WAV.
Telegram не подключается: Проверь токен/Chat ID.
Макрос не детектирует: Настрой оверлей точно на бар рыбалки (чтобы не задевало ГУИ с наживками), подкрути Kp/Kd.
